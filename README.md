
# Titanic EDA Project

Exploratory Data Analysis on the Titanic dataset using Python (Pandas & NumPy).

## Key Findings
- Female passengers had a 74% survival rate vs. 19% for males
- 1st class passengers survived at 63%, compared to 24% in 3rd class
- Passengers with small families (2-4 members) had the highest survival rates
- Fare distribution was highly right-skewed (mean $32 vs median $14)

## Tools Used
- Python, Pandas, NumPy
- Jupyter Notebook (via Kaggle)
